# AIDRA Models

This directory contains AI model weights for vessel detection.

## Download pre-trained models

```bash
./scripts/download-models.sh
```

This downloads:
- `yolov8n.pt` — YOLOv8 nano base model (~6 MB)
- `yolov8s.pt` — YOLOv8 small model (~22 MB)
- `yolov8n.onnx` — YOLOv8 nano exported to ONNX

## Fine-tuned models

After fine-tuning with xView3-SAR (see `scripts/fine-tune.py`):
- `yolov8n-sar.pt` — Fine-tuned on SAR vessel detection
- `yolov8n-sar.onnx` — ONNX export of fine-tuned model

## SAR ship detection (vesselTracker-style)

From [hewitleo/sar-ship-detection-yolov8](https://huggingface.co/hewitleo/sar-ship-detection-yolov8) (Apache 2.0):
- `vesseltracker-sar-yolov8.pt` — YOLOv8m trained on Sentinel-1 SAR imagery (~50 MB)
  - Architecture: YOLOv8m (depth=0.67, width=0.75), 25.9M params, 295 modules
  - Classes: 1 (`ship`)
  - Input: 640x640 RGB (SAR single-channel replicated to 3 channels)
  - Inspired by [carlossantamarizq/vesselTracker](https://github.com/carlossantamarizq/vesselTracker) (YOLOv5s, weights not published)

## Compressed variants

Generated by the compression module (`src/models/compression/`):
- `yolov8n-sar-fp16.onnx` — FP16 quantization (~3 MB)
- `yolov8n-sar-int8-dynamic.pt` — Dynamic INT8 (~1.5 MB)
- `yolov8n-sar-int8-static.onnx` — Static INT8 with calibration (~1.5 MB)
- `yolov8n-sar-pruned30.pt` — 30% pruned (~4.2 MB)
- `yolov8n-sar-pruned50.pt` — 50% pruned (~3 MB)

## Note

Model files (.pt, .onnx) are gitignored due to their size.
Run the download script after cloning the repository.
